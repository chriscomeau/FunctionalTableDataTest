# Building and Running


**In the Terminal:**


* You can install Carthage with [Homebrew](http://brew.sh/) using the following command:

        $ brew update
        $ brew install carthage
      

* Update frameworks with Carthage 
         
        $ carthage update


* Open `FunctionalTableDataTest.xcodeproj` in Xcode.
